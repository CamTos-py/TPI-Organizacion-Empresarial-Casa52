> Organización Empresarial

> Universidad Tecnológica Nacional

 Descripción

Proyecto de automatización de gestión de turnos mediante un chatbot simulado en un código desarrollado en python y documentos .csv com base de datos en la cual se apoya el sistema.

 Funcionalidades

- Validación de ID
- Ver turno semanal
- Solicitar cambio de turno
- Pedir día libre
- Manejo de errores

 Base de datos simulada

- empleados.csv
- turnos.csv
- solicitudes.csv
